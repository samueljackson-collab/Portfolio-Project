# Kuiper Sys/NetAdmin Interview Cheat Sheet - Enhanced Edition

**Quick Reference for Interview Day**

---

## How to Use This Cheat Sheet

**Night Before Interview:**
- Read through all 20 questions
- For each: attempt to explain without looking at answer
- Draw diagrams on paper for questions marked ⚠️ DIAGRAM

**2 Hours Before:**
- Review "Core Answer" column only
- Do warm-up exercise at bottom
- Have portfolio bookmarks ready

**During Interview:**
- Don't memorize - understand the concepts
- Use Feynman method to explain simply first, then dive into technical details
- Reference your portfolio when relevant ("I actually implemented this in my homelab...")

---

## Enhanced Interview Question Table

### Columns Explained:
1. **Q#** - Question number
2. **Topic/Question** - What you'll be asked
3. **Core Answer** - Key points to hit (30-60 second answer)
4. **Feynman (Explain Simply)** - How to explain to non-technical person
5. **Acronyms** - All acronyms used, spelled out
6. **Diagram Needed?** - Whether to draw on whiteboard
7. **Portfolio Artifact** - What project shows this
8. **Risk** - What can go wrong (operational risk)
9. **Timebox** - How long to spend answering
10. **Owner** - Who typically asks this
