# Service Deployment Runbook

## Creating LXC Containers

1. Log into Proxmox UI
2. Create CT with Ubuntu template
3. Configure resources and network
4. Start and configure

## Deploying Docker Services

1. Install Docker
2. Create docker-compose.yml
3. Deploy with `docker compose up -d`

## Configuring Nginx Proxy

1. Access NPM at 10.0.10.40:81
2. Add proxy host
3. Configure SSL
4. Add DNS entry
