Hardware

Proxmox Host

Dell PowerEdge R720 - Primary virtualization host running Proxmox VE 8.1.4

  • CPU: 2x Intel Xeon E5-2670 (8 cores, 16 threads each = 32 threads total)
  • RAM: 128GB DDR3 ECC (8x 16GB modules)
  • Storage: 4x 1TB Samsung 870 EVO SSD in RAID10 (2TB usable)
  • Network: Dual 10Gb SFP+ ports (bonded for redundancy)
  • Management: iDRAC 7 Enterprise for remote management
  • Power: Dual 750W redundant power supplies

TrueNAS Storage Server

Supermicro Custom Build - Dedicated network-attached storage

  • CPU: Intel Xeon E3-1230v3 (4 cores, 8 threads)
  • RAM: 32GB ECC (critical for ZFS integrity)
  • Storage: 6x 4TB WD Red HDD in RAIDZ2 configuration (16TB usable)
  • Cache: 2x 120GB SSD for ZFS L2ARC
  • Network: Dual 1Gb Ethernet (LACP bonded)

Network Architecture

The network is segmented into four VLANs for security and traffic isolation:

Service Architecture

Services are designed with the following principles:

  • Separation of Concerns: Each VM runs a single primary service
  • Centralized Database: PostgreSQL VM serves all applications
  • Reverse Proxy: All external traffic routes through Nginx Proxy Manager
  • Monitoring: Prometheus + Grafana collect metrics from all services
  • High Availability: Automated backups with Proxmox Backup Server

External Access

Services are exposed to the internet via Cloudflare + Nginx Proxy Manager:

Internet ‚Üí Cloudflare (DNS + DDoS) ‚Üí Public IP ‚Üí Router (Port Forward 80/443) ‚Üì Nginx Proxy Manager (192.168.40.25) - SSL Termination ‚Üì Backend Services (Wiki.js, Home Assistant, Immich)

Backup Strategy

Multiple backup layers ensure data protection:

  • VM Backups: Proxmox Backup Server runs daily at 2:00 AM
  • Database Backups: PostgreSQL pg_dump runs hourly
  • Storage Snapshots: TrueNAS ZFS snapshots every hour (retained 7 days)
  • Offsite Backup: Weekly rclone sync to Backblaze B2

Monitoring & Observability

Comprehensive monitoring stack provides visibility into infrastructure health:

  • Prometheus: Metrics collection from all VMs and services
  • Grafana: Visualization dashboards for infrastructure metrics
  • Loki + Promtail: Centralized log aggregation
  • Alertmanager: Alerts sent to Slack and email
