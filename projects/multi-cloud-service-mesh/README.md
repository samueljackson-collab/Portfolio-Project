# Multi-Cloud Service Mesh

**Category:** Networking          **Status:** Complete

## Overview
Istio multi-cluster mesh with mTLS and observability.

## Tech Stack
Istio, Kubernetes, Prometheus

## Running Locally
1. Clone the repo and navigate to `multi-cloud-service-mesh`.
2. Follow the service-specific README or docker-compose file to start dependencies.
3. Run the test suite to validate changes.
