Portfolio Sample Frontend

This lightweight static frontend demonstrates the local developer experience for the portfolio export. It pairs with the FastAPI backend exposed at http://localhost:8080 and the supporting PostgreSQL/Redis services defined in docker-compose.yml.

Smoke Test Targets

  • http://localhost:3000 (this page)
  • http://localhost:8080/health
  • http://localhost:8080/health/db
  • http://localhost:8080/health/redis

Use scripts/test/smoke-tests.sh to validate all endpoints at once.

Next Steps

Replace this static implementation with your production frontend and extend the backend service with domain logic. The sample stack uses idiomatic environment variables so it can be dropped into Kubernetes or other orchestrators.
