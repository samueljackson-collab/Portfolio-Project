## ğŸ”§ Required Tools

### 1. Git (Version Control System)

**What is it?**
Git is the core version control system that tracks changes in your code.

**Version Required:** 2.30 or later

**Installation:**
- We'll cover installation in detail in [Lesson 3](./03-git-installation-windows.md)
- Download from [git-scm.com](https://git-scm.com)

**How to check if installed:**
```bash
git --version
# Should output: git version 2.30.0 or higher
```

### 2. Text Editor

**Recommended: Visual Studio Code**
- Free, powerful, and widely used
- Built-in Git integration
- Download: [code.visualstudio.com](https://code.visualstudio.com)

**Alternatives:**
- **Sublime Text** - Lightweight and fast
- **Atom** - GitHub's own editor
- **Notepad++** (Windows) - Simple and reliable
- **vim/emacs** - For experienced terminal users

**Why not Word/Google Docs?**
These add hidden formatting that breaks code. Always use plain text editors.

### 3. Terminal/Command Line

**What is it?**
A text-based interface to interact with your computer and Git.

**Windows:**
- **Git Bash** (Recommended) - Installed with Git
- **PowerShell** - Built into Windows
- **Command Prompt** - Basic but works
- **Windows Terminal** - Modern, powerful option

**macOS:**
- **Terminal** - Built-in, found in Applications/Utilities
- **iTerm2** - Popular alternative with more features

**Linux:**
- Your distribution's terminal (GNOME Terminal, Konsole, etc.)

### 4. Web Browser

**Any modern browser works:**
- Chrome/Chromium
- Firefox
- Safari
- Edge

**Why you need it:**
- Access GitHub web interface
- View documentation
- Manage repositories online
- Create pull requests
