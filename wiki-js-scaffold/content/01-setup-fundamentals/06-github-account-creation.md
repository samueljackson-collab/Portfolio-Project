# GitHub Account Creation & Setup

## üåü Create Your GitHub Account

### Step 1: Sign Up

1. Visit [github.com](https://github.com)
2. Click **Sign up** (top right)
3. Enter your email address
4. Create a strong password
5. Choose a username (this will be your GitHub identity!)

### Username Tips:
- ‚úÖ Professional and memorable
- ‚úÖ Consistent with other platforms
- ‚úÖ Easy to spell and remember
- ‚ùå Avoid numbers or special characters if possible

**Examples:**
- Good: `john-smith`, `janedev`, `mikecodes`
- Avoid: `johnsmith12345`, `c00ld3v3l0p3r`

### Step 2: Verify Your Account

1. Solve the puzzle (CAPTCHA)
2. Click **Create account**
3. Check your email for verification code
4. Enter the 6-digit code

### Step 3: Personalize Your Experience

GitHub will ask some questions:
- What kind of work do you do?
- What are you interested in?
- How experienced are you?

Answer honestly - this helps GitHub recommend relevant content.

## üé® Optimize Your Profile

A professional profile helps with job hunting and networking!

### Access Your Profile

1. Click your profile photo (top right)
2. Select **Your profile**
3. Click **Edit profile**

### Essential Profile Elements

**Profile Picture**
- Professional headshot or logo
- 400x400 pixels minimum
- Clear and recognizable

**Bio**
- One line describing what you do
- Examples:
  - "Full-stack developer passionate about open source"
  - "Data scientist learning machine learning"
  - "Student exploring web development"

**Location**
- City, Country (optional but helps with networking)

**Website/Blog**
- Personal portfolio
- LinkedIn profile
- Blog or Medium

**Social Accounts**
- LinkedIn
- Twitter
- Personal website

## ‚öôÔ∏è Configure Settings

### Email Settings

**Settings ‚Üí Emails**

1. **Primary email**: Your main contact email
2. **Backup email**: Add secondary email (optional)
3. **Email preferences**:
   - ‚úÖ Keep "Receive updates about GitHub" checked
   - Configure notification preferences

**Privacy:**
- ‚òëÔ∏è "Keep my email addresses private"
- ‚òëÔ∏è "Block command line pushes that expose my email"

### Notification Settings

**Settings ‚Üí Notifications**

Configure how you want to be notified:
- Email notifications
- Web notifications
- Frequency (realtime, hourly, daily)

**Recommendations for beginners:**
- Enable email notifications for repositories you watch
- Disable automated notifications until you understand them

### Security Settings

**Settings ‚Üí Password and authentication**

1. **Two-Factor Authentication (2FA)**
   - Click "Enable two-factor authentication"
   - Use an authenticator app (recommended):
     - Google Authenticator
     - Authy
     - Microsoft Authenticator
   - Or SMS (less secure but better than nothing)
   - **Save recovery codes** in a safe place!

2. **Sessions**
   - Review active sessions
   - Sign out suspicious sessions

### SSH Keys

**Settings ‚Üí SSH and GPG keys**

Add the SSH key you created in the previous lesson:
- See: [SSH Key Setup](./05-ssh-key-setup.md)

## üêõ Common Issues

### Verification Email Not Received

1. Check spam/junk folder
2. Verify email address is correct
3. Resend verification email: Settings ‚Üí Emails
4. Try a different email provider if problems persist

### Username Already Taken

- Add your profession: `john-developer`
- Add your location: `john-seattle`
- Add numbers sparingly: `johnsmith21`

### Can't Enable 2FA

- Ensure authenticator app is installed
- Time on phone matches time on computer
- Try different authenticator app
- Use SMS as backup method
