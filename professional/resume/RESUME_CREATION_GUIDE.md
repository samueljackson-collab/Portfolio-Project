# Resume Creation Guide

## Overview

Create targeted resumes for different career paths.

## Variants

### 1. SDE/DevOps
- Focus: IaC, CI/CD, Monitoring
- Projects: PRJ-SDE-001, PRJ-SDE-002
- Skills: Terraform, Prometheus, Docker

### 2. Cloud Engineer
- Focus: AWS, Architecture
- Projects: PRJ-SDE-001
- Skills: AWS, Terraform, CloudFormation

### 3. Network Engineer
- Focus: Network design, VLANs
- Projects: PRJ-HOME-001
- Skills: Networking, Firewalls, VLANs

### 4. QA Engineer
- Focus: Testing, Automation
- Projects: Test infrastructure
- Skills: pytest, Testing

### 5. Security
- Focus: Security, Compliance
- Projects: Network segmentation
- Skills: Security monitoring

## Best Practices

- Use action verbs
- Quantify achievements
- Link to portfolio
- Tailor to job description
