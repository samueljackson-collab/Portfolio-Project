## Navigation Flow for Recruiters

### Path 1: Quick Overview
1. Start at `README.md` → See all 25 projects listed
2. Click any project → Opens project-specific documentation
3. Browse technical details, architecture, evidence

### Path 2: Deep Dive into Homelab Infrastructure
1. `README.md` → Click "Homelab Projects"
2. Open `PRJ-HOME-001/README.md` → Network & Security setup
3. Click `assets/documentation/network-architecture.md` → Technical diagrams
4. Click `assets/runbooks/network-deployment-runbook.md` → Operational procedures
5. Click `assets/configs/pfsense-config.xml` → Configuration evidence

### Path 3: Security & Operations Expertise
1. `README.md` → Click "Security Automation"
2. Open `projects/p19-security-automation/RUNBOOK.md`
3. Review incident response procedures (P0-P3 classification)
4. Navigate to related observability project via cross-reference
5. Open `projects/p20-observability/RUNBOOK.md`
6. Review monitoring and alerting setup

### Path 4: AI/ML Capabilities
1. `README.md` → Click "AI/ML Projects"
2. Open `projects/07-aiml-automation/PRJ-AIML-002/RUNBOOK.md`
3. Click `docs/architecture.md` → System design
4. Click `docs/deployment.md` → Implementation details
